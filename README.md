## Sobre este proyecto

Actualmente, las mujeres enfrentan un problema social de inseguridad al tener que transitar por distintas zonas peligrosas; existen rutas consideradas de alto riesgo, especialmente durante la noche, y esto aumenta la probabilidad de que se ocasionen incidentes que afecten su seguridad y bienestar. Ante la problemática planteada, se desea desarrollar una aplicación que contribuya a mejorar la seguridad de las mujeres, permitiendo encontrar rutas que sean seguras y alertar sobre zonas poco confiables. Además, la aplicación cuenta con funciones de geolocalización, monitoreo en tiempo real y alertas de emergencia que permitan brindar una respuesta rápida ante situaciones de riesgo.
